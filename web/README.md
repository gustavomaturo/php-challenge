# Project Execution

- install [NodeJS](https://nodejs.org/en/) e [NPM](https://www.npmjs.com/);
- Clone the repository;
- Enter the folder via the terminal;
- Download depedencies **npm install**;
- Run the project **http-server**;
- Alter urlApi in js/const.js
